# Full Stack E-Commerce Platform

A full-stack e-commerce web application built using the **PERN stack (PostgreSQL, Express.js, React, Node.js)**.  
The platform allows users to browse products, manage carts, authenticate securely, and place orders through a responsive interface.

## Live Demo
Frontend: https://e-commerce-hazel-chi.vercel.app  
Backend: https://github.com/Romel164255/E-commerce

---

# Features

### User Features
- User registration and login
- Secure authentication using **JWT**
- Browse and search products
- Add/remove items from cart
- Place orders and manage purchases

### Admin Features
- Product management (Create, Update, Delete)
- Manage orders
- Manage users
- Secure admin authorization

### System Features
- RESTful API architecture
- Protected routes using middleware
- Database integration with PostgreSQL
- Responsive UI built with React
